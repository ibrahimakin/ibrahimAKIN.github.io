Privacy Policy

Extension: Search YouTube  |  Developer: ibrahim AKIN

Overview

Search YouTube (the “Extension”) is provided by ibrahim AKIN at no cost and is intended to be used as is. The Extension uses the YouTube API to retrieve search results.

We respect your privacy. This Extension does not collect, store, or share any personal or device data. There are no accounts, analytics, advertising SDKs, trackers, or cookies.

Information Collection and Use

The Extension only uses the YouTube API to obtain search results. No data from your device or browsing activity is transmitted to the developer or stored by the developer. Any requests to the YouTube API are for the sole purpose of providing the Extension’s core functionality.

Data Sharing

Because we do not collect data, we do not share any data with third parties, service providers, advertisers, affiliates, or analytics platforms.

Cookies and Tracking

The Extension does not use cookies or similar tracking technologies.

Children’s Privacy

The Extension is suitable for users of all ages. We do not knowingly collect personal information from children under 13, or from any user.

Security

Since the Extension does not handle user data, there is no personal information stored on our servers that requires protection. To ensure authenticity and integrity, download the Extension only from official extension stores where we publish it.

Changes to This Policy

If the Extension changes to begin collecting data, this policy will be updated before such changes take effect. We will revise the Last updated date below and, where required, provide notice within the Extension or via the extension store listing.
